Triangle Classifier (C)

This project consists of an algorithm developed in the **C** language that receives three integer values representing the sides of a triangle and performs its validation and classification.

## 📋 Challenge Description

The program reads three integer sides and executes the following steps:

1. **Validation**: Checks if the sides form a valid triangle. A triangle is valid if each side is smaller than the sum of the other two.
2. **Classification**:

* **Equilateral**: All 3 sides are equal.
* **Isosceles**: Only 2 sides are equal.
* **Scalene**: All 3 sides are different.
